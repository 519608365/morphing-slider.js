# react-gulp-sample

## Overview
This is a sample webapp using React.js and gulp.

## Tools
* gulp
* React.js
* Sass

## Getting Started
Clone the repositoty.  
    
    git clone git@github.com:mildsummer/react-gulp-sample
    

Set up the modules.  
    
    npm install
    

Run a gulp.  
    
    gulp
    

Please enter the following URL to launch the browser.  
    
    http://localhost:8080
