# react-gulp-sample

## Overview
This is a sample webapp using React.js and gulp.

## Tools
* gulp
* React.js
* Sass

## Getting Started
Clone the repositoty.  
    
    git clone git@github.com:mildsummer/react-gulp-sample
    

Set up the modules.  
    
    npm install
    
Install the bower components.  
    
    bower install
    
Run a gulp. Then, the browser will start up.
    
    gulp
